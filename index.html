H2 within a div with a class of x_title - test 11/2

This is the content container. It is as div with the class of x_content.

The layout for this type of panel is as follows:

  • div container with the class of x_panel
    • div with class of x_title
      • h2 with the Panel Title
      • Another ul which handles the collapse feature
    • div with the class of x_content

Please inspect the code if needed. I believe we'll use the x_title class for areas outside of the x_panel as well to keep things looking the same. These styles will translate to the legend element in forms as well. Below is a blank panel to make copying easier. Place this panel directly into a container with the Bootstrap grid class.
